# Veltrixa AI Assistant 🤖

Veltrixa AI Assistant is a Retrieval-Augmented Generation (RAG) based AI chatbot that answers questions from PDF documents.

This project demonstrates how AI can retrieve information from documents and generate context-aware answers using embeddings and a language model.

The PDF used in this project represents **Veltrixa**, which is a **virtual company created for demonstration purposes**. The document contains sample company information used to test the RAG pipeline.

---

## 🚀 Features

- Ask questions from PDF documents
- Retrieval-Augmented Generation (RAG)
- Semantic search using embeddings
- Streamlit based interactive UI
- Context-aware answers from document content

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Sentence Transformers
- Vector Store for embeddings

## ⚙️ Installation

Clone the repository:

git clone https://github.com/yourusername/Veltrixa_AI_Assistant.git

Go to the project folder:

cd Veltrixa_AI_Assistant

Install dependencies:

pip install -r requirements.txt

---

## ▶️ Run the Project

Start the Streamlit application:

streamlit run Ui.py

After running the command, open the local URL shown in the terminal.

## 📄 Example Questions

You can ask questions like:

- What does Veltrixa company do?
- What services does Veltrixa provide?
